JavaScript

ORIGEN

Fue creado por Brendan Eich en 1995. Originalmente se llamaba "Mocha" y luego "LiveScript", pero finalmente se le cambió el nombre a JavaScript.

LENGUAJE

Es un lenguaje de scripting que se utiliza principalmente para agregar interactividad a las páginas web. Eso significa que se ejecuta en el navegador web del usuario, en lugar de en un servidor.

SINTAXIS

Tiene una sintaxis similar a otros lenguajes de programación como C y Java. Utiliza palabras clave como var, if, else, for, while para controlar el flujo de ejecución del código.

ORINETACIÓN

Es un lenguaje orientado a objetos, lo que significa que se basa en la idea de objetos que contienen datos y métodos. Los objetos se pueden crear utilizando la palabra clave new o mediante la creación de objetos literales.

USOS

Se usa para interactividad en las páginas web, validación, comunicaciones, desarrollo de aplicaciones entre muchas otras cosas

DOM

Es una representación de un documento HTML como una estructura de árbol de nodos. Cada elemento, atributo y texto del documento se representa como un nodo.

Proporciona una interfaz de programación (API) para acceder y manipular el contenido, la estructura y el estilo de un documento HTML.

Se puede usar JavaScript para modificar el DOM, por ejemplo, agregar, eliminar o cambiar elementos, atributos y estilos.

Permite detectar y responder a eventos del usuario, como clics, pulsaciones de teclas y cambios de tamaño de la ventana.

Puedes navegar por el árbol del DOM para encontrar elementos específicos o para acceder a los nodos padre, hijo o hermanos de un nodo.

Es fundamental para crear páginas web dinámicas e interactivas, permitiendo que los usuarios interactúen con el contenido y modifiquen la apariencia de la página web en tiempo real.

REFERENCIAR

Se puede usar document.getElementById('id') para obtener una referencia a un elemento HTML con un ID específico. Esto es útil para manipular un elemento único en la página.

Se puede usar
document.
getElementsByClassName('clase') para obtener una colección de elementos HTML que comparten una clase específica. Esto es útil para manipular varios elementos con la misma clase.

Referenciar elementos en JavaScript permite manipular su contenido, estilo y comportamiento. Puedes cambiar su texto, agregar o eliminar atributos, aplicar estilos CSS, agregar eventos y mucho más.

STYLE

USO

La propiedad style permite acceder y modificar los estilos CSS de un elemento HTML desde JavaScript.

SINTAXIS

Los nombres de las propiedades CSS se escriben con la sintaxis de camello en JavaScript, por ejemplo, backgroundColor en lugar de background-color.

VALORES

Se pueden asignar valores a las propiedades style de un elemento usando el operador de asignación =, por ejemplo, elemento.style.backgroundColor = 'red'.

MANIPULACIÓN

Puedes usar la propiedad style para cambiar el color, el tamaño de fuente, la posición, la visualización y otras propiedades CSS de un elemento.

INTERACCIÓN

Es fundamental para crear páginas web dinámicas e interactivas, permitiendo que los usuarios interactúen con el contenido y modifiquen la apariencia de la página web en tiempo real.

CLASSLIST

ACCESO

La propiedad classList permite acceder y manipular las clases CSS de un elemento HTML desde JavaScript.

COLECCIÓN

classList es una colección de objetos que representan las clases CSS aplicadas a un elemento.

MÉTODOS

classList ofrece métodos como add(), remove(), toggle(), contains(), replace(), item(), entre otros, para gestionar las clases de un elemento.

AGREGAR CLASES

elemento.classList.add('clase') agrega una nueva clase al elemento.

ELIMINAR CLASES

elemento.classList.remove('clase') elimina una clase del elemento.

CONTROL DINÁMICO

La propiedad classList te permite controlar dinámicamente las clases CSS de un elemento en respuesta a eventos o cambios en la página web.

EVENTOS

Son acciones que ocurren en una página web, como clics del ratón, pulsaciones de teclas, movimientos del ratón, carga de la página, entre otros.

Permite detectar y responder a estos eventos, permitiendo que tu página web sea interactiva y dinámica.

Se puede usar la función addEventListener() para agregar un manejador de eventos a un elemento, que se ejecutará cuando ocurra el evento.

Los manejadores de eventos son funciones que se ejecutan cuando se produce un evento específico.

Los eventos te permiten controlar el flujo de la ejecución del código en respuesta a la interacción del usuario, creando una experiencia web más dinámica e interactiva.

IF ELSE

Son estructuras te permiten ejecutar diferentes bloques de código dependiendo de si una condición se cumple o no.

La condición dentro de la estructura if se evalúa como verdadera o falsa. Si es verdadera, se ejecuta el código dentro del bloque if.

Si la condición del if es falsa, se ejecuta el código dentro del bloque else (si es que existe).

El if-else se usa para ejecutar un bloque de código si la condición es verdadera y otro bloque de código si la condición es falsa.

Else se usa para ejecutar un bloque de código solo si ninguna de las condiciones anteriores se cumple.

VIEWPORT

Es fundamental para el diseño web moderno, ya que permite que las páginas web se vean correctamente en una variedad de dispositivos.

Es la región visible de la página web en un dispositivo. Es decir, es el área que el usuario puede ver en su pantalla en un momento dado.

La etiqueta meta name="viewport" permite definir el ancho, alto y escala del viewport. Esto es crucial para el diseño responsive, que se adapta a diferentes tamaños de pantalla.

También puede utilizarse para detectar el tamaño del viewport y ajustar el contenido en consecuencia.

@MEDIA

@media es una regla CSS que permite aplicar estilos diferentes a tu página web dependiendo de las características del dispositivo o del navegador del usuario.

Se puede usar @media para crear diseños responsivos que se adapten a diferentes tamaños de pantalla, orientaciones, resoluciones y otros factores.

@media se define dentro de las hojas de estilo CSS y no directamente en JavaScript.
